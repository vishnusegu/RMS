# Car rentals system

This project is used to book cars online and if need they and cancle and update their profile in case of need and some other features.

## Technology Stack Used :

- Backednd : NodeJS,
- Database : Mongodb
- FrontEnd : JS, Html, css
- Framework : Express

## Usage

1. Clone and navigate to this project directory

2. Install the dependencies

   ```bash
   npm install
   ```

3. Run the node/express http server
   ```bash
   node app.js
   ```
4. Open the url `http://localhost:3000` in browser for seeing the result
